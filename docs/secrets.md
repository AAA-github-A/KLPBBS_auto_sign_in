# [KLPBBS_auto_sign_in](https://github.com/xyz8848/KLPBBS_auto_sign_in)
## Secrets

|名称|值|备注|
|-|-|-|
|USERNAME| |苦力怕论坛用户名|
|PASSWORD| |苦力怕论坛密码|
|DEBUG|`0`, `1`|调试模式（0=关闭，1=开启）|
|MAIL_ENABLE|`0`, `1`|签到后邮件提示（0=关闭，1=开启）|
|MAIL_HOST| |SMTP 服务器|
|MAIL_PORT| |SMTP 端口|
|MAIL_USERNAME| |邮箱账号|
|MAIL_PASSWORD| |邮箱密码|
|MAIL_TO| |用于接收通知的邮箱|
